A simple expense and income tracking application built using React and Vite.
It allows users to add, view, and manage their daily transactions with an easy interface.

Features

.Add both expenses and income.
.Select category, date, and optional notes for each transaction.
.View all transactions in a clean list.
.See total spending and income in a summary section.
.Light and dark theme support using Context API.
.All data is stored locally using localStorage, so nothing is lost on refresh.
.Responsive layout that works on both mobile and desktop.
.Basic charts to show spending visually.

Tech Stack

.React (Vite)
.JavaScript (ES6+)
.Tailwind CSS
.LocalStorage (for storing all expenses)
.Chart.js (for charts and graphs)
.react-chartjs-2 (for charts and graphs)
.Libraries Used
.React – main UI framework
.Vite – fast development bundler
.Chart.js – used for doughnut and line charts
.Tailwind CSS – styling the UI
